# Webサイト
- Appleの製品紹介サイトみたいな、スクロールで背景画像が切り替わるスライド
- 背景がリッチな演出によって入れ替わり、文字を引き立てる構成
- 文字は、PC版では右側、SP版では下に配置することで、PC版、SP版にそれぞれ最適化する
- 文字は全文を入れるのではなく、見出しや象徴的なところだけ入れる
- 途中、実際にこのWebサイトをAI生成している様子を動画で差し込みたい

## 演出
「ドリフト」演出は、単なるページの切り替えではなく、「境界が溶けて混ざり合う」ような流動的な視覚体験を指します。具体的には、以下のようなイメージです。
1. 「インクの滲み」のような境界の遷移
セクションが切り替わる際、パキッとした直線で画面が分かれるのではなく、水面にインクを落としたときのように、次のセクションの色や形がじわじわと侵食してくる演出です。

イメージ: スクロールに合わせて、前のページの背景色が波打ちながら、次のページの色と混ざり合い、新しい模様を作り出します。

2. 「陽炎（かげろう）」や「液状」の歪み
固定された「設計図」から外れる不安定さや自由さを表現するために、スクロールの速度に応じてコンテンツ全体が少し歪む（ディスプレイスメント・マッピング）演出です。

イメージ: まっすぐなテキストや図形が、スクロール中だけ水中に沈んでいるかのようにゆらゆらと揺れ、動きが止まるとゆっくりと元の形に収束します。

3. 「色彩の漂流（グラデーションの動的変化）」
セクションごとに背景色を固定するのではなく、スクロール量（Scroll Progress）に同期して、背景のグラデーションがゆっくりと変容し続けます。

イメージ: プレゼンの進行に合わせて、深い霧のようなグレーから、光が差し込むような暖色へと、境界なく色が「漂流」していくような感覚を与えます。

4. 物理演算を用いた「重なりのズレ」
各要素が、スクロールに対して少しずつ遅れて動く「パララックス（視差効果）」に、わずかな「揺れ」や「慣性」を加えます。

イメージ: 画面上のテキストや画像が、まるで水面に浮かんでいる筏（いかだ）のように、スクロールの勢いでわずかに慣性を持って動き、少し遅れて定位置に止まるような動きです。

## 視覚構成
1. 「文字を引き立たせる」視覚構成
背景を単なる画像ではなく「空気感（テクスチャ）」として扱い、その上に力強いタイポグラフィを配置することで、原稿にある言葉の重みが際立ちます。

背景の処理: 原稿に登場する「農場の土の色」「錆止め塗装の赤」「深い霧」などをイメージした抽象的な背景を用意します。スクロールに応じて、これらの色が「ドリフト（漂流）」するように混ざり合っていく演出です。

文字の動き: スクロールと連動して、文字が単に上下に動くのではなく、わずかに奥行きを持って迫ってくる（あるいは遠ざかる）ようなパララックス効果を加えます。これにより、読者は言葉の海を泳いでいるような感覚になります。

2. 「ここぞという図解」の差し込みポイント
原稿の中で、視覚的な補助があると一気に理解が深まるポイントが3箇所あります。そこだけ「動く図解」をインラインで差し込むのが効果的です。

「感じる力」の変容: 「泥のついた手袋でスマホを操作する」という具体例を、単なる写真ではなく、ユーザーの操作性がどう変わったかを示すビフォー・アフターの対比図として提示します。

「美術の授業の観察構造」: 原稿にある「先生・生徒・作品」の三者の関係性を、スクロールに合わせて視線が動く三角形のダイアグラムで表現します。

「凸凹と山」: 画面の下から「AI（海）」がせり上がってきて、「凹（谷）」を埋めていくアニメーション。最終的に「凸（山）」だけが島のように残る様子を視覚化します。

3. 実装のヒント：Cursor (Opus) への追加指示
「文字を引き立たせる」ための具体的なコード構成として、以下のような指示をCursorに与えると、よりリッチな質感が得られます。

指示案:
「背景の div に backdrop-filter: blur(20px) をかけた半透明のレイヤーを重ねて、その上のテキストが背景のグラデーションを透かしつつも、くっきりと読めるように調整して。また、セクションの切り替わりで framer-motion の mix-blend-mode: difference を一瞬適用して、境界が溶けるような視覚効果を狙いたい。」

この構成のメリット
集中力の維持: ずっと同じトーンの背景ではなく、言葉に合わせて背景の「質感」が変わるため、30分の発表でも聴衆が飽きません。

メッセージの純度: 余計な装飾を削ぎ落とし、「言葉」と「ここぞという図解」だけに絞ることで、原稿にある「問いを立てる力」という本質的なテーマがストレートに伝わります。
