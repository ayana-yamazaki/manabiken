# ペルソナ & スライド構成

## ペルソナ設定

### Persona A：中堅の中学校美術教師「田中先生」
- 42歳女性、教職17年目、公立中学校の美術科教諭
- 対話型鑑賞の実践経験あり。学び研には数年前から参加
- **最大の痛み**：管理職や保護者から「美術って何の役に立つの？」と聞かれたとき、感覚的にはわかっているのに、論理的に返す言葉がない。職員会議で美術の時間数が議題に上がるたびに胃が痛い
- **フォーラムに求めていること**：「外の世界の人が、美術で育った力をどう使っているか」の具体的な証言。明日の職員室で使える言葉
- **刺さるポイント**：レポサクの事例（「感じる力」がグッドデザイン賞金賞に直結した話）、「対話による鑑賞」が社会でそのまま使われている接続

### Persona B：若手の小学校図工専科「木村先生」
- 29歳男性、教職5年目、小学校の図工専科
- 子どもが制作に没頭する姿に手応えを感じているが、それを「学び」として言語化できていない
- **最大の痛み**：生成AIで子どもが一瞬で「上手な絵」を出力する場面を目の当たりにし、「描く技術を教える意味はあるのか」というアイデンティティ不安がある。STEAM教育の文脈で「美術のA」をどう位置づけるか模索中
- **フォーラムに求めていること**：AI時代でも揺るがない美術教育の本質的な価値の言語化。自分より先にAIの波を被った人の、生々しい体験談
- **刺さるポイント**：デザイナー自身が「仕事をAIに奪われた」リアルな告白からの反転、プロトタイプ型の生き方（図工の授業がまさにそれ）

### Persona C：教育委員会の指導主事「佐藤さん」
- 55歳男性、元中学校美術教師→現在は県の指導主事
- カリキュラム編成と教員研修の設計が仕事。美術の時間数削減の圧力に対峙する立場
- **最大の痛み**：美術の価値を制度の中で守りたいが、「エビデンス」や「産業界の声」が不足している。数値で語りにくい教科だからこそ、外部からの説得力ある証言が喉から手が出るほど欲しい
- **フォーラムに求めていること**：政策レベルで引用できるストーリーとフレーム。「AI時代に美術が必要」という産業界の実務家からの明確なメッセージ
- **刺さるポイント**：「問いを立てる力」という明確なフレーム、美術の授業にしかない「観察の構造」の論理的な整理、凸凹の話（特別支援・インクルーシブ教育との接続可能性）

## スライド構成（35分・全28枚）

### 第0部：導入（3分）— 全ペルソナの警戒を解く

| # | スライド | 狙い |
|---|---------|------|
| 1 | **タイトル**「問いを立てる力は、どこで育つのか——デザインの現場から見た美術教育」 | |
| 2 | **自己紹介**：プロダクトデザイナー／美大出身／製造業・農業・建設業の現場でアプリを設計。「美術の先生ではありません。美術の授業を受けた側の人間です」 | 「外部ゲストへの警戒」を解く。「美術をわかっている人」であることを早めに示す |
| 3 | **本日のメッセージ（先出し）**：「AI時代に最も必要とされる力は、美術の授業が育てうる力だ」＋3つの観点を予告 | **C** が「引用できるフレーム」の予感を得る。全員が聞く構えになる |

### 第1部：背景——AIは「答え」を奪った（5分）— Bが最も食いつくパート

| # | スライド | 狙い |
|---|---------|------|
| 4 | **私の仕事の紹介**：UI/UXデザインとは何か。レポサクの画面写真。グッドデザイン賞金賞（ベスト20）受賞 | 専門家としての信頼性を確立。**C** に「この人の話は引用に耐える」と思わせる |
| 5 | **Before/After**：「丸一日かけて1画面を描いていた」→「2026年、AIに言葉で伝えるだけで動く画面が出る」 | デザイナーの仕事の変化を端的に。全員に衝撃を与えるスライド |
| 6 | **「デザイナーと名乗っていいかわからない」**：自分の肩書きへの戸惑い。納品物が絵→コードに変わった | **B** の「図工を教える意味があるのか」という不安と共振。「この人も同じ不安を抱えている」 |
| 7 | **転換点**：「AIは"答え"を出すのが圧倒的に速い。しかし"問い"を立てることは人間の仕事だ」→ その力を育てうるのが美術教育ではないか | 全ペルソナに対する核心の提示。ここから3つの理由へ |

### 第2部：理由① 美術は「感じる力」を育てている（8分）— Aが最も食いつくパート

| # | スライド | 狙い |
|---|---------|------|
| 8 | **レポサク開発の壁**：「俺らにはIT無理だから」農家さんの拒絶。怒ってスマホを机に投げるエピソード | ストーリーテリングの起点。全員の注意を引く |
| 9 | **現場で感じ取る**：畑を借りて2年間通った。日光でスマホが見えない。手袋で操作できない。シニアスマホ教室にも通い始めた。「マップアプリが何度聞いてもわからない」。言葉にならない切なさとプライド | 「現場に行って感じた」プロセスの可視化＋感情のレイヤーの深掘り。**A** が「これは鑑賞の授業と同じだ」と感じ始める |
| 10 | **問いの転換**：「使いやすいアプリ」ではなく「農家さんが誇りを持って使える体験」へ → 操作そのものを無くす設計、IT用語を隠す、畑の上に余計なものを被せない | 「感じる」が設計の方角を変えた証明。**C** が「これはエビデンスとして使える」と手応えを得る |
| 11 | **口コミで全国200社へ**：「これ知ってる？ かっこいいだろ」農家さんの誇り | 成果の提示。「感じる力」→ ビジネス成果への接続 |
| 12 | **美術教育との接続**：「私がこのプロセスを最初に体験したのは、対話による鑑賞の授業だった」。否定しない場で他者の見方に触れる → 自分の見え方が変わる → 問いが生まれる | **A** にとって最大の山場。「自分がやっている対話型鑑賞が、グッドデザイン賞金賞につながっている」。職員室で使える接続 |
| 13 | **まとめ①**：感じる力 → 問いを立てる力 → 価値を生む力（図解） | 構造を整理。**C** が研修資料に使えるフレーム |

### 第3部：理由② 美術は「プロトタイプ型の生き方」を教えている（8分）— B・Aに刺さるパート

| # | スライド | 狙い |
|---|---------|------|
| 14 | **2つの人生モデル（図解）**：「設計図型」vs「プロトタイプ型」。設計図型＝正解を決めてから動く。プロトタイプ型＝小さく試して、感じて、ドリフトする | 全員に刺さる二項対立。わかりやすい構造 |
| 15 | **美大での椅子制作**：スケッチ何十枚 → 実寸モデルで座る →「なんか違う」→ 壊す → 作る → またドリフト | 「美術教育の延長線上で仕事をしている」実例。**B** が「図工でやっていることだ」と確信 |
| 16 | **友人A氏のエピソード**：優秀な公務員が転職できない。選択肢が多すぎるのではなく「これがいい」と感じる力が弱っている | 全ペルソナが「こういう人いる」と共感する日常エピソード |
| 17 | **AI時代に設計図型が加速する**：AIの自信に満ちた即答が「正解は一回で出るもの」という思い込みを強める | **B** の「子どもがAIに頼る」現実感と接続 |
| 18 | **美術教育との接続**：「美術の授業はドリフトする練習だ。粘土をこね、色を塗り、"なんか違う"と壊してやり直す。あの感覚が、社会に出てからの生き方になった」 | **A** と **B** の両方に「自分の授業がやっていることだ」という確信を渡す |
| 19 | **まとめ②**：設計図型の限界 → プロトタイプ型の生存戦略 → 美術がその練習場である（図解） | 構造整理 |

### 第4部：理由③ 美術の授業には「観察の場」としての構造がある（6分）— A・Cに深く刺さるパート

| # | スライド | 狙い |
|---|---------|------|
| 20 | **教室の風景**：「算数の時間はずっと下を向いていたのに、美術の時間になると手が止まらなくなる子」 | **A** が「あの子だ」と具体的な生徒の顔を思い浮かべる |
| 21 | **美術の構造的な特異性**：テストの答案ではなくプロセスを見られる。没頭の仕方、やり直しの仕方に、その子の関心と思考の癖が滲む | **C** にとって「美術固有の教育的価値」の制度的な論拠になる |
| 22 | **山と谷の図**：AIが谷（苦手）を埋める → 残るのはその人だけの山（得意）。しかし子どもは自分の山に気づけない → 先生だけが気づける | このプレゼンで最もエモーショナルなスライド。全ペルソナの感情を動かす |
| 23 | **まとめ③**：美術の教室 = 他教科では見えない「山」が顔を出す場所 → 先生が最初に気づける | 構造整理 |

### 第5部：結論——先生方へのメッセージ（5分）— 全ペルソナの感情を収束

| # | スライド | 狙い |
|---|---------|------|
| 24 | **3つの理由の総括**（1枚で図示）：感じる力 / プロトタイプ型 / 観察の場 | **C** が「この3点を研修で引用する」と決められるフレーム |
| 25 | **私自身の凸凹**：テストで最下位と上位が同居。遅刻・忘れ物・バイトはクビ。でも凹があるから農家さんの痛みがわかった。「凹が共鳴装置になった」 | **A** が自分の生徒と重ねる。人間味が出て距離が縮まる |
| 26 | **この原稿自体が実践**：AIとの共著。論理的にまとめるのが苦手な私の凹をAIが埋めた。残ったのは私の山だけ | メタ構造の開示。笑いと納得 |
| 27 | **最後の問いかけ**：「もし美術の時間にだけ別人のように動き出す子がいたら、それがその子だけの力であることを、本人に伝えてほしい」 | 全ペルソナへの最終メッセージ。**A** と **B** は具体的な生徒の顔が浮かんでいる。**C** は「この言葉を研修で使う」と思っている |
| 28 | **締め**：「先生方の教室から飛び立つ子どもたちと、いつか一緒に仕事がしたい」 | 余韻を残して終了 |
