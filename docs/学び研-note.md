## 学び研リサーチメモ

### 基本情報
- 正式名称：美術による学び研究会（通称「学び研」）
- 設立：2008年2月、高知で開催された第3回「美術鑑賞教育フォーラム」にて、高知大学の上野行一氏が始動を提案
- 学会化：2018年11月、10年の節目に「日本・美術による学び学会」（JSLA: Japan Society for Learning through Art）として正式設立
- 事務局：北翔大学 教育文化学部（北海道江別市）
- 代表：山崎正明（※筆者の父）
- 会員規模：2009年に150人 → 2011年に300人 → 2013年に500人超

### 理念・特徴
- 従来の「教師→学習者」の一方通行的な教育観から脱却し、学習者の視点から教育を捉え直す
- 対話型鑑賞（VTC、VTS、意味生成的アプローチなど）を重視
- 美術教育における「創造力」「感性」「学習者間の相互作用」「体験・身体性」を大切にする
- 教育学・美術・哲学・心理学・社会学など分野横断的

### 主な活動
- 学会誌『美術による学び』の編集・発行
- 全国大会、ミニ大会（大分・北海道・秋田等）、リサーチフォーラムの開催
- オンライン講座「対話型鑑賞を学ぶ」BASIC / BEGINNER コース
- シンポジウム・実践発表・研究討議

### フォーラムについて
- 2025年度にリサーチフォーラムの開催告知あり（artedu.jp/researchforum）
- フォーラムでは実践発表やシンポジウムが行われ、冊子に発表原稿がまとめられる
- 元々の出発点自体が「美術鑑賞教育フォーラム」であり、フォーラム形式の研究交流が学び研のDNA

### 公式サイト
- 学び研：https://www.art.gr.jp/
- 学会：https://www.artedu.jp/

## フォーラムのオーディエンス特性

### 主な構成層
- **図画工作・美術科の教員**（小・中・高）が最大多数。現場で日々授業を行っている実践者層
- **大学の美術科教育学の研究者**（上野行一氏、山崎正明氏など）。理論と実践の橋渡しを志向
- **教育委員会・指導主事**。カリキュラム編成や研修設計の立場から参加
- **教科書出版関係者**（日本文教出版など）。教材開発への示唆を求めて参加
- **他教科からのSTEAM関心層**（少数）。教科横断的な学びの可能性を探っている

### 年齢・キャリア層
- 中堅〜ベテラン教員（教職10年以上）が中心と推定。若手は少数派
- 「学び研」の設立経緯（2008年〜）から、長年参加しているコア層が存在
- 対話型鑑賞（VTS等）の実践経験者が多く、「正解のない学び」への理解は深い

### 聴衆が抱える課題感・関心
1. **「美術って何を学べるの？」という社会的な問いへの回答を求めている**
   - 管理職・保護者・他教科教員に美術の価値を説明する言葉が足りないという焦り
2. **AI時代における美術教育の存在意義への不安**
   - 生成AIが「きれいなもの」を瞬時に作れる現在、技能教育の根拠が揺らいでいる
3. **STEAMの"A"としての美術の位置づけ**
   - 他教科との連携で美術がどう貢献できるかの具体例を渇望
4. **「感覚から思考へ」のプロセス化**
   - 制作して終わりではなく、振り返り・言語化を通じた深い学びへの転換を模索中
5. **多忙な現場との折り合い**
   - やる気は高いが、時間的・制度的制約の中でどう越境的実践を増やすか

### 聴衆の心理的傾向
- **味方意識が強い場**：美術教育の価値を信じている者同士の集まりなので、基本的に「共感ベース」で聴く
- **外部ゲストへの期待と警戒**：産業界からの登壇者に対して「美術をわかってくれるか？」という軽い緊張感がある
- **実践に落とせるかどうかを重視**：抽象論だけでは物足りず、「明日の授業でどう使えるか」を無意識に求めている

### 知能特性（2E：twice-exceptional）

  - 低い側（類似80・絵画配列85）→「できない人」の気持ちが体感としてわかる。知識や観察ではなく、自分自身が「できない」経験を持っているから
  - 高い側（積み木模様120・理解125）→ その体感を構造化し、UIという形に翻訳できる
- グッドデザイン賞金賞の農家アプリは、この2Eの特性から生まれた。農家のおじいちゃんがスマホにブチギレていた怒りの裏にある「言葉にできない切なさ・プライド」を感じ取れたのは、自分自身に「できなさ」があるから

### 原稿の核となる気づき
- 「現場に足繁く通ったからユーザーの気持ちがわかる」のではなく、「知能が低い部分があるから、できない人の気持ちがわかる」という自己認識
- AIは完璧すぎて弱点がない → だから「できない人の気持ち」を体感として知らない → 人間の不完全さこそが、AIにできないことを可能にする
- 美術の時間は「知能の時間」ではない。感じて、試すだけでいい → 学校の中で唯一の「感じる練習」の時間
- 「感じる力」が原稿全体を貫く軸：AIは考えるのが得意だが感じることができない / 選択肢が無限でも「これがいい」と感じられれば動ける / 美術は感じる練習をしている
